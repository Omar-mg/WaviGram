# packages/shared — Shared types and constants (Phase 2)

This package will hold cross-app types, Zod schemas, and constants that are shared between the frontend and backend.

**Planned contents:**
- `src/types/` — User, Message, Conversation, Post, Story, etc.
- `src/schemas/` — Zod request/response schemas
- `src/constants/` — Shared enums (message status, user role, etc.)

This package will be created in **Phase 2**.
