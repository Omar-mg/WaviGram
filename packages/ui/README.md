# packages/ui — Shared component library (Phase 3)

This package will be the home of the WaviGram design system components once they stabilize. In Phase 1, components live in `apps/web/src/components/ui/` for fast iteration.

Once components are stable (after Phase 2), we'll extract them here so that `apps/api` (e.g. for email templates) and any future apps can consume them.
